| 課題2 .行列席の計算　　　　　　　　　　  |
|                                   |
=====================================

1. ファイル
  ・readme.txt	: このファイル
  ・kadai.c		: mainソースファイル
  ・NAbasic.c		: 行列データ処理関数のソースファイル 
  ・NAbasic.h		: NAbasic用ヘッダファイル
  ・k2-input1.csv	: 入力データCSVファイル1
  ・k2-input2.csv	: 入力データCSVファイル2

2. 処理内容
入力ファイル1と入力ファイル2より行列を読み取り，行列積を計算します．

入力ファイル1より読み込んだ行列：A
入力ファイル2より読み込んだ行列：x
計算式：A・x = (出力)

3. コンパイル方法
"kadai.c"と"NAbasic.c"をコンパイルする．
例）Embarcadero C++ Compiler
  > bcc32c kadai.c NAbasic.c

4. 実行方法
コンパイルによって生成された実行ファイルに引数として入力CSVファイルを渡して実行する．
例）実行ファイル：kadai.exe，入力ファイル1："k2-input1.csv"，入力ファイル2："k2-input1.csv"の場合
  > kadai.exe k2-input1.csv k2-input2.csv

5．入力ファイル
ファイル形式：CSV
内容：
　入力ファイル 1：行列（左）
　入力ファイル 2：行列（右）
