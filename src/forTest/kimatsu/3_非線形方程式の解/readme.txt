[コンパイル方法]
kadai3に加えて添付された1つの関数群ソースファイルをコンパイルしてください。

Embarcadero C++の場合
> bcc32c week3.c NAbasic.c

[実行方法]
コンパイルした後、生成された実行ファイルを実行してください。その際、引数として入力データのファイルを指定してください。

実行ファイル：kadai3.exe、入力データ：k3-input.csv の場合
> kadai3.exe k3-input.csv

[開発環境]
OS      :   Microsoft Windows 11 Home
Ver.    :   10.0.22621 ビルド 22621
CPU     :	AMD Ryzen 3 5300U
物理RAM :   8GB
