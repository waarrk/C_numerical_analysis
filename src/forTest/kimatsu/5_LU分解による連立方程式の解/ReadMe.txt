課題5. LU分解を用いた連立方程式の解法

□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□□

1．ファイル
  readme.txt	: このファイル
  kadai5.c		: 課題のプログラムのソースファイル
  NAbasic.c		: 行列データ処理関数のソースファイル 
  NAbasic.h		: NAbasic用ヘッダファイル
  k5-input1.csv	: 入力データCSVファイル
  k5-input2.csv	: 入力データCSVファイル

2．処理内容
入力ファイルから行列とベクトルを読み取り，LU分解を用いて連立方程式の解を求めます。

例）
入力ファイル1：行列A
入力ファイル2：ベクトルb
解：行列x

Ax=b
を計算します。出力は順にL行列，U行列，P行列，解xです。

3．コンパイル方法
"kadai5.c"と"NAbasic.c"をコンパイルする．
例）Embarcadero C++ Compiler
  > bcc32c kadai5.c NAbasic.c

4．実行方法
コンパイルによって生成された実行ファイルに引数として入力CSVファイルを渡して実行する．
例）実行ファイル：kadai5.exe，入力ファイル1："k5-input1.csv"，入力ファイル2："k5-input2.csv"の場合
  > kadai5.exe k5-input1.csv k5-input2.csv

5．入力ファイル
ファイル形式：CSV
内容：実数行列（ベクトル）
