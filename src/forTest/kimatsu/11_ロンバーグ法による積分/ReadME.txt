　ロンバーグ法による積分　ReadMe
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

[はじめに]
・このプログラムはロンバーグ法による積分のプログラムです。
・同梱されたプログラム等を用いて発生したいかなる問題も自己責任の上でご利用ください。

[関連ファイル]
・k11-input.csv         :確認用データ
・romberg.c(.h)         :ロンバーグ法による積分に関する関数群
・NAbasic.c(.h)         :数値解析用基本関数群
・week11.c              :メイン関数を含むコアプログラム

[コンパイル方法]
・week11.c，romberg.c，NAbasic.cをまとめてコンパイルしてください。

Embarcadero C++ 7.30 for Win32の場合
> bcc32c ./week11.c ./romberg.c ./NAbasic.c

[実行方法]
・生成された実行ファイルを以下の形式で実行してください。
> (実行ファイル) （入力ファイル）（分割数）

例）
> ./week11.exe k11-input.csv 5

[出力内容]
・入力された点データを上から1つずつ用いて値推定を行います。4点の値を入力した場合，1-4点による補間の結果を順に表示します。

OS          :Microsoft Windows 11 Home

Compiler    :Embarcadero C++ 7.30 for Win32
